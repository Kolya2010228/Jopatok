# Jopatok - Локальный TikTok 🎬

Android-приложение для просмотра видео из локальных папок в стиле TikTok.

![Android Build](https://github.com/Kolya2010228/Jopatok/workflows/Android%20Build%20&%20Release/badge.svg)

## 📱 Особенности

- **Вертикальные свайпы** - переключение между видео как в TikTok
- **Выбор папок** - выбирайте любые папки с видео на устройстве
- **Поддержка форматов** - MP4, MKV, AVI, MOV, WebM, 3GP
- **Минималистичный дизайн** - в стиле TikTok
- **Android 5.0+** - поддержка API 21 и выше

## 🚀 Быстрый старт

### Сборка APK

```bash
git clone https://github.com/YOUR_USERNAME/Jopatok.git
cd Jopatok
./gradlew assembleDebug
```

APK файл будет в: `app/build/outputs/apk/debug/app-debug.apk`

### Установка

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
