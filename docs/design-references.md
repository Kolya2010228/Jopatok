# 🎨 Дизайн-референсы Jopatok

## Интерфейс приложения

### Главный экран (лента видео)

```
┌─────────────────────────────────┐
│                                 │
│         ┌───────────┐           │
│         │           │           │
│         │   VIDEO   │   ❤️      │  <- Кнопки справа
│         │           │   Like    │
│         │           │           │
│         │           │   💬      │
│         │           │   Comment │
│         │           │           │
│         │           │   ↗️      │
│         │           │   Share   │
│         └───────────┘           │
│                                 │
│  📹 Video Title                 │
│  📁 Folder Name                 │
│                                 │
└─────────────────────────────────┘
          ▲ свайп вверх/вниз
```

### Экран выбора папок

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│            📁                   │
│                                 │
│   Добро пожаловать в Jopatok!   │
│                                 │
│ Выберите папки с видео и        │
│ наслаждайтесь просмотром        │
│                                 │
│    ┌─────────────────┐          │
│    │  Выбрать папку  │          │
│    └─────────────────┘          │
│                                 │
│ Можно выбрать несколько папок   │
│                                 │
└─────────────────────────────────┘
```

## Цветовая схема

- **Фон**: #000000 (чёрный)
- **Текст**: #FFFFFF (белый)
- **Акцент**: #6200EE (фиолетовый)
- **Вторичный**: #03DAC5 (бирюзовый)

## Элементы управления

| Элемент | Описание |
|---------|----------|
| Свайп вверх | Следующее видео |
| Свайп вниз | Предыдущее видео |
| Тап по видео | Пауза/воспроизведение |
| Двойной тап | Лайк |
| Кнопка ❤️ | Лайк видео |
| Кнопка 💬 | Комментарии (будущая функция) |
| Кнопка ↗️ | Поделиться (будущая функция) |

## Анимации

- Плавное переключение между видео
- Эффект нажатия на кнопки (scale animation)
- Индикатор загрузки (buffering)
